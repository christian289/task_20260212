# 과제 기본 조건

---

- [필수]로 표시된 항목은 모두 작성해야만 합니다.
- [필수]로 나열한 부분 외에는 자유롭게 구성 / 작성 하면 되며, 여러 의미로 해석이 가능한 경우는 본인이 원하는대로 해석 후 코드를 작성하면 됩니다.
- 프로젝트 실행 / 확인시 특수한 세팅과 파라미터가 필요한 경우 같이 작성해 주세요.
- 실제 서비스로 배포 한다는 생각으로 실무에 최대한 가깝게 구현해주세요.
- 지원자가 원하는 편한 방법으로 작성하여 개발해 주시면 됩니다.
- 지원자의 코드를 비롯하여 github 사용, 코딩 습관, 코드 구조, 컨벤션 예외 처리, 테스트 코드 작성 등 여러 종합적인 부분들을 확인하게 됩니다.

# 과제 문제

---

- CLO 에서는 직원들의 긴급 연락망 구축을 위해서 시스템을 만들려고 합니다.
- 주어진 정보를 빠르게 확인할 수 있도록 화면 구성은 Front 팀에 담당 했고, Backend 팀은 그에 맞춰진 API를 만들어야 합니다.
- 주어진 조건은 다음과 같습니다

1. [필수] 직원의 기본 연락 정보를 알 수 있어야 합니다
2. 직원 정보 추가를 위하여 csv, json 2가지 형식으로 사용 됩니다
   [필수] csv 파일 업로드시 작동
   [필수] json 파일 업로드시 작동
   [필수] body 에 csv 직접 입력시 작동
   [필수] body 에 json 직접 입력시 작동

csv 예제 (필수 데이터 이외에 행/열을 더 추가/삭제 하셔도 됩니다)
반드시 csv 형식은 아래의 필수 데이터 요소가 포함이 되어야 합니다.

김철수, charles@clovf.com 01075312468, 2018.03.07
박영희, matilda@clovf.com 01087654321, , 2021.04.28
홍길동, kildong-hong@clovf.com 01012345678, 2015.08.15

json 예제 (필수 데이터 이외에 행/열을 더 추가/삭제 하셔도 됩니다)
반드시 json 형식은 아래의 필수 데이터 요소가 포함이 되어야 합니다.

[
{"name":"김클로", "email":"clo@clovf.com", "tel":"010-1111-2424","joined":"2012-01-05"},
{"name":"박마블", "email":"md@clovf.com", "tel":"010-3535-7979","joined":"2013-07-01"},
{"name":"홍커넥", "email":"connect@clovf.com", "tel":"010-8531-7942","joined":"2019-12-05"},
]

## [필수]로 구현 해야 하는 API Endpoints

口 직원들의 기본 연락 정보를 얻을 수 있습니다
GET /api/employee?page={page}&pageSize={pageSize}
Response ( 200 Ok, application/json )
Front-end 쪽에서 작업이 가능하도록, 전체 데이터를 보여주고 페이징 (paging) 가능하도록 출력 해주세요.

口 직원의 기본 연락 정보를 얻을 수 있습니다
GET /api/employee/{name}
Response (성공시 200 Ok, application/json )
Front-end 쪽에서 보여줄 수 있도록, 요청한 name과 일치하는 직원의 상세 연락정보 반환 해주세요.

口 직원의 기본 연락 정보를 추가 할 수 있습니다
POST /api/employee
Response (성공시 201 Created)
Front-end 쪽에서 요구하는 아래 조건을 모두 만족해야 합니다

- <input type=file> 을 통해 csv , json 파일 업로드시 작동 합니다
- <textarea></textarea> 에서 직접 데이터 입력 했을 경우에도 작동 됩니다

# 요구사항

## Required

- 필수 사용 Framework
  - .NET 8 이상 (C#) (은 CLO의 선호하는 언어입니다)
- 최신 버전에서 지원하는 스펙을 잘 이용해 주시기를 바랍니다

### 필수 구현

- CQRS 패턴 형태로 코드 만들기
- 올바르게 동작 하는 성공 또는 실패 케이스에 대한 테스트 코드를 작성
